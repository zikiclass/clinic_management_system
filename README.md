## About Clinic Management System

This Clinic Management System is built with nice features. The features are;

- Patient make online appointment.
- Patient confirm attendance to appointment.
- Doctor view booking slots.
- Follow up patient medical history.
- Send prescription to pharmacist.
- Pharmacist determine current inventory of medicine when it's running low.
- View waiting list of patients waiting for their prescriptions.
